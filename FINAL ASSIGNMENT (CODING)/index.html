Projects

1. Packaging Design

Project Brief: In this Project I had to create a typeface and then use the typeface in packaging design which suits the typeface. As per my thinking, a shoe brand was perfect fit for my typeface as it was very sporty, energetic, and geometric typeface.

2. Book Design

Project Brief: In this Project I had to create a book design which contains 70% of text and 30% of illustrations. The topic for book design was weight loss and the age group was 13-18 years old. I had chosen a vibrant and eye-catchy colour palette as it was a book for childrens. Inside the book I had tried to incorporate different illustrations inside the numbers. Overall, it was a good project, it taught me some design skills and also improved my design thinking.

3. The Museum Room Project

Project Brief: This was a Museum Room Group Project. In this project we had to design a interior space of a Museum Room according to the artifacts and other objects of artistic, cultural, historical, or scientific importance displayed in the museum. We were given the theme Luxury to design the room, so we gave a dark blue colour to the wall with a golden embossed texture.

4. Food Brochure

Project Brief: In this project we had to create a food brochure for any city in India. I had chosen Hyderabad because it offers an unparalleled culinary experience that makes it an ideal choice for food brochure. Also I had taken color palette which gives a vibe of Hyderabad and also the color palette mathces with the dishes from hyderabad.

5. Soda Can Packaging Design

Project Brief: In this project we had to re-design soda can packaging according to Renaissance art movement. We had to incorporate renaissance art features in soda can packaging. I had chosen Coca-Cola for this re-design. I incorporated motifs, patterns and design style from renaissance movement. The outcome came out really well. It was looking as if Coca-Cola was there in Renaissance movement.
